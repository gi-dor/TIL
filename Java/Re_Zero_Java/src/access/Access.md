- private : 모든 외부의 호출을 막는다
- default : 같은 패키지 안에서 호출은 허용한다
- protected : 같은 패키지 안에서 호출은 허용 , 패키지가 달라도 상속관계의 호출은 허용한다
- public : 모든 외부 호출을 허용한다


package-private
- 접근제어자를 명시하지 않은면 같은 패키지 안에서 호출을 허용하는 `default`  접근제어자가 적용된다

접근제어자 사용위치
- 접근제어자는 필드와 메서드 , 생성자에서 사용된다
- 클래스 레벨에도 일부 접근 제어자를 사용하 수 있다
```java
public class Speaker {
    private int volume;
    public void volumeUp(){}
}
```

접근제어자의 핵심은 속성과 기능을 외부로부터 숨기는 것이다
- `private`  나의 클래스 안으로 속성과 기능을 숨길 때 사용 , 외부클래스에서 호출할 수 없다
- `default` 패키지 안으로 속성과 기능을 숨길 때 사용 , 외부 패키지에서 해당 기능을 호출할 수 없다
- `protected` 상속관계로 속성과 기능을 숨길때 사용 , 상속관계가 아닌 곳에서 해당 기능을 호출할 수 없다
- `public` 기능을 숨기지 않고 어디서든 호출할 수 있게 공개한다
