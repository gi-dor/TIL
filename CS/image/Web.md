### 1. 웹 서버 Web Server

- HTTP 기반으로 동작
- 정적 리소스 제공 , 기타 부가 기능
- 정적(파일) HTML CSS JS 이미지 영상
- Ex. NGINX , APACHE

### 2. 웹 애플리케이션 서버 WAS - Web Application Server

- HTTP 기반으로동작
- 웹서버 기능 포함 + 정적 리소스 제공
- 프로그램 코드를 실행해서 애플리케이션 로직 수행
  - 동적 HTML , HTTP API(JSON)
  - 서블릿 JSP 스프링MVC
- Ex. 톰캣 , Jetty , Undertow
